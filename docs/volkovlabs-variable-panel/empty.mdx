# Empty value in Multi-value

:::info Version
The empty value option was introduced in version 2.0.0.
:::

This feature works for the following Minimize and Button display modes.

<Image
  title="Empty Value allowed for the Minimize and Button Display modes."
  src="/img/blog/2023-10-19-variable-panel-2.0.0/empty-value.png"
/>

Before this feature was introduced, a user had selected an item in a Multi-value variable, there was no way to unselect all values.

<Image
  title="Multi-value setting for dashboard variables."
  src="/img/blog/2023-10-19-variable-panel-2.0.0/multi-value.png"
/>

For this feature to work, you must enable the <b>Empty Value</b> parameter in the Variable section.

<Image
  title="When Empty value is enabled, a user can unselect all variable values."
  src="/img/blog/2023-10-19-variable-panel-2.0.0/empty-all.png"
/>
