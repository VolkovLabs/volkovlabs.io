# Input Text Box

:::info Version

Variable Panel supports the input text box starting from version 1.6.0. We added a community-requested feature allowing input value for the Text box variables.

:::

The input text box is supported in the table and minimized display modes.

If you select the table and enable the feature, the input can be sticky and follow when scrolling the dashboard.
