# Variable Panel

The Variable panel allows you to update dependent and multi-variables as a Table and Tree View.

:::info Development

The Variable Panel is under development and Grafana team review.
:::

<Image
  title="Variable panel allows selecting devices by Country, State and City."
  src="/img/plugins/volkovlabs-variable-panel/dashboard.png"
/>

## Requirements

- **Grafana 9** and **Grafana 10** are required for major version 1.

## Getting Started

The Variable panel is not included in the Grafana Catalog. It can be downloaded directly from GitHub.

## Highlights

- Allows updating dashboard variables as a Table and Tree View.
- Displays statuses based on thresholds from data sources.
- Supports single and multi-value variables with the All option.
- Allows filtering values.
- Supports follow when scrolling (sticky).
