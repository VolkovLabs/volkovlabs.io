# Variables

Dashboard and Global variables will be replaced automatically in

- URL for Initial and Update requests.
- Header Parameters' values.
- Payload of the Update Request.
- JavaScript code.

Three types of variables are discussed in the [Grafana Crash Course](/grafana/variables).

## JavaScript code

```js
const formIcon = elements.find((element) => element.id === "icon");
formIcon.value = "$IconVar";
```
