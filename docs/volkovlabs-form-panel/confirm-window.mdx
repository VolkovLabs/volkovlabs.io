# Confirmation Window

The Confirmation window, if enabled, is shown after a user clicks the submit button to highlight all changes made in the Data Manipulation form.

:::info Version

Confirmation window labels are supported since version 3.3.0.

:::

You can customize all labels using the **Confirmation Window** section.

<Image
  title="Customize confirmation window label names."
  src="/img/blog/2023-12-03-form-panel-3.3.0/confirm.png"
/>

Since version 4.0.0 displayed values in confirmation window can be configurable with new option `Display Values`. You can choose `All values` or `Updated Only`.

<Image
  title="Confirmation window Display values option"
  src="/img/plugins/volkovlabs-form-panel/confirmation-window-values.png"
/>
