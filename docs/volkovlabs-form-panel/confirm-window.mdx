# Confirmation Window

The Confirmation window, if enabled, is shown after a user clicks the submit button to highlight all changes made in the Data Manipulation form.

:::info Version

Confirmation window labels are supported since version 3.3.0.

:::

You can customize all labels using the **Confirmation Window** section.
