# Custom Requests

Data Manipulation panel allows you to create your own Initial and Update requests using Custom Code.

## Initial Request

Select Initial Request as `-` and set Custom Code:

<Code
  url="https://github.com/VolkovLabs/volkovlabs.io/blob/main/docs/volkovlabs-form-panel/js/initial-request.js"
  language="js"
/>

### Initial values

To support `Highlight changed values` and `Require Confirmation` the Custom Code should use `setInitial({})` function to update `initial` values:

```javascript
setInitial({ value: 99, name: "Test" });
```

## Update Request

Select Update Request as `-` and set Custom Code. Depending on the selected Payload options it will add all or only updated values.
