# API Server for MySQL database

JSON API Server on Node.js allows to retrieve and update value using Data Manipulation panel in the MySQL database.

## Features

- CORS enabled.
- Customizable using Environment Variables.
- Has checks and error handling.
- Includes SQL statements, Dockerfile and README file to get started.
