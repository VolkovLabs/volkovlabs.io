# Control Panel

The Data Manipulation plugin allows the creation of control panels with multiple sections and the same Id.

<Image
  title="Control Panel created using the Data Manipulation panel."
  src="/img/plugins/volkovlabs-form-panel/control.png"
/>

Additional features:

- Read-only elements support lookup options similar to Radio and Select elements to display labels for the current state.
- Update Request allows the inclusion of only updated values in the Payload.
- Ask to confirm updated values before the Update Request.
