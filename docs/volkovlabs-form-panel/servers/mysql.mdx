# API Server for MySQL

API Server on Node.js allows retrieval and update value using the Data Manipulation panel in the MySQL database.

## Features

- CORS enabled.
- Customizable using Environment Variables.
- Has checks and error handling.
- Includes [SQL statement](https://github.com/VolkovLabs/volkovlabs-form-panel/blob/main/server-mysql/feedbacks.sql), [Dockerfile](https://github.com/VolkovLabs/volkovlabs-form-panel/blob/main/server-mysql/Dockerfile) and [README](https://github.com/VolkovLabs/volkovlabs-form-panel/blob/main/server-mysql/README.md) to get started.
