# JSON API Server

JSON API Server on Node.js enables retrieving and updating JSON objects to test Data Manipulation Panel.

## Features

- CORS enabled.
- Update requests can be `PATCH`, `POST`, or `PUT`.
- Includes the [Docker image](https://github.com/VolkovLabs/volkovlabs-form-panel/blob/main/server-json/Dockerfile) to get started.
