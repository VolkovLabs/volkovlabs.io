# JSON API Server

JSON API Server on Node.js allows retrieval and update of JSON objects to test the Data Manipulation panel.

## Features

- CORS enabled.
- Update requests can be PATCH, POST, and PUT.
- Includes [Dockerfile](https://github.com/VolkovLabs/volkovlabs-form-panel/blob/main/server-json/Dockerfile) to get started.
