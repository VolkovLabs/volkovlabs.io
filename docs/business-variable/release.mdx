# Release Notes

Follow the development of the Business Variable panel with these release notes, detailing new features, enhancements, fixes, and breaking changes across all versions.

## 3.9.0 (2025-04-08)

This release focuses on improving user experience with custom messages for edge cases, a minimized view for Tree View layout, and compatibility with Grafana 11.6.0.

## 3.8.0 (2025-03-26)

This version introduces enhancements to the release workflow, removes outdated dependencies, and improves panel usability with resize behavior for sticky headers.

## 3.7.0 (2025-02-13)

A significant update with improvements to variable handling, UI enhancements for sticky headers and favorites, and support for Grafana 11.4.0, along with new components like the Date Picker.

## 3.6.0 (2024-10-24)

This release enhances usability with new components like Date Time Selector, improved Table view behavior, and compatibility updates to Grafana 11.3.0.

## 3.5.0 (2024-10-02)

Focused on refining the Minimize view for select variables and ensuring compatibility with Grafana 11.2.2 through dependency updates.

## 3.4.0 (2024-08-23)

This update introduces options for reordering tabs and updates local storage for favorites, alongside compatibility with Grafana 11.1.4.

<BlogPost id="variable-panel-3.4.0-20240823" />

## 3.3.0 (2024-08-14)

Enhancements include support for data source favorites and UI improvements like scrollbars for Button view.

## 3.2.0 (2024-07-09)

This release focuses on user education with updated tutorials and performance optimizations through Docker workflows and new styling options in Table view.

## 3.1.0 (2024-06-06)

Improvements to user interaction with collapse/expand options, enhanced selection behaviors, and compatibility with Grafana 11.0.0.

## 3.0.0 (2024-05-09)

:::warning Breaking Changes
Requires Grafana 10 or 11
:::

A major release with breaking changes requiring Grafana 10 or 11, introducing new views like Slider View, performance enhancements, and extensive UI improvements.
