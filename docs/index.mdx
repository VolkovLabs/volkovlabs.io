# Business Suite

We specialize in creating plugins that enable you to visualize your business data on the Grafana platform.

<Image title="Business Suite for Grafana." src="/img/honeycomb.png" />

| Plugin                                                      | Description                                                |
| ----------------------------------------------------------- | ---------------------------------------------------------- |
| [Business Calendar Panel](volkovlabs-calendar-panel)        | Display events and change time range.                      |
| [Business Charts Panel](volkovlabs-echarts-panel)           | Powerful visualizations powered by Apache ECharts.         |
| [Business Forms Panel](volkovlabs-form-panel)               | Insert, update application data, and modify configuration. |
| [Business Image/PDF Panel](volkovlabs-image-panel)          | Display Base64 encoded media files.                        |
| [Business News Data Source](volkovlabs-rss-datasource)      | Visualize RSS/Atom feeds using Dynamic Text.               |
| [Business Text Panel](volkovlabs-dynamictext-panel)         | Dynamic, data-driven text.                                 |
| [Business Variable Panel](volkovlabs-variable-panel)        | Update dashboard variables.                                |
| [Environment Data Source](volkovlabs-env-datasource)        | Environment variables on your dashboard.                   |
| [Grafana HTTP API Data Source](volkovlabs-grapi-datasource) | Connect to Local and Remote Grafana instances.             |
| [Static Data Source](volkovlabs-static-datasource)          | Store and emulate your data.                               |
