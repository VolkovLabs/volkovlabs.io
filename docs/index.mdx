The Business Suite for Grafana is an open-source collection of plugins tailored for business-focused users. It delivers versatile solutions to meet a wide range of business needs, enabling you to transform dashboards into powerful, actionable tools.

- **Business-Driven**: Designed with business processes in mind.
- **Comprehensive**: Addresses diverse requirements with a unified suite.
- **Open Source**: Freely accessible and community-supported.

## Plugins

Explore the Business Suite’s powerful plugins, each crafted to enhance your Grafana experience:

| Plugin                                   | Type        | Description                                                                       |
| ---------------------------------------- | ----------- | --------------------------------------------------------------------------------- |
| [Business Calendar](business-calendar)   | Panel       | Showcases events in an elegant calendar view, sourced from various data inputs.   |
| [Business Charts](business-charts)       | Panel       | Brings Apache ECharts’ dynamic charts and graphs to your Grafana dashboards.      |
| [Business Forms](business-forms)         | Panel       | Enables data entry, updates, and configuration changes directly within Grafana.   |
| [Business Input](business-input)         | Data Source | Emulates and stores data using JavaScript and LLM for flexible data management.   |
| [Business Links](business-links)         | Panel       | Streamlines navigation with external links, dashboards, and dropdowns.            |
| [Business Media](business-media)         | Panel       | Combines multimedia formats for a customized visual experience in Grafana.        |
| [Business News](business-news)           | Data Source | Fetches RSS and Atom feeds for visualization with Business Text and other panels. |
| [Business Satellite](business-satellite) | Data Source | Retrieves data from local or remote Grafana instances via HTTP API.               |
| [Business Table](business-table)         | Panel       | Simplifies data presentation with intuitive table layouts.                        |
| [Business Text](business-text)           | Panel       | Transforms plain text and tables into visually engaging information cards.        |
| [Business Variable](business-variable)   | Panel       | Adds flexible dashboard filters in a movable, standalone panel.                   |
