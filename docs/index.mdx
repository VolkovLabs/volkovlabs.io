# Business Suite for Grafana

- Designed specifically for users focused on the business side of the process.
- Provides solutions for wide spectrum of business requirements.
- Open source.

<Image
  title="Business Suite for Grafana created and actively maintained by Volkov Labs."
  src="/img/honeycomb.png"
/>

| Plugin                                            | Type        | Description                                                |
| ------------------------------------------------- | ----------- | ---------------------------------------------------------- |
| [Business Calendar](volkovlabs-calendar-panel)    | Panel       | Display events and change time range.                      |
| [Business Charts](volkovlabs-echarts-panel)       | Panel       | Powerful visualizations powered by Apache ECharts.         |
| [Business Forms](volkovlabs-form-panel)           | Panel       | Insert, update application data, and modify configuration. |
| [Business Input](volkovlabs-static-datasource)    | Data source | Store and create your data.                                |
| [Business Media](volkovlabs-image-panel)          | Panel       | Display Base64 encoded media files.                        |
| [Business News](volkovlabs-rss-datasource)        | Data source | Visualize RSS/Atom feeds using Business Text.              |
| [Business Satellite](volkovlabs-grapi-datasource) | Data source | Connect to Local and Remote Grafana instances.             |
| [Business Text](volkovlabs-dynamictext-panel)     | Panel       | Dynamic, data-driven text.                                 |
| [Business Variable](volkovlabs-variable-panel)    | Panel       | Update dashboard variables.                                |
