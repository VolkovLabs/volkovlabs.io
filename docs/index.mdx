## Plugins

| Plugin                                   | Type        | Description                                                                                                                     |
| ---------------------------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------- |
| [Business Calendar](business-calendar)   | Panel       | Displays events in a stylish calendar format. The data can come from various data sources.                                      |
| [Business Charts](business-charts)       | Panel       | Integrates charts and graphs created by the Apache ECharts library into your Grafana dashboard.                                 |
| [Business Forms](business-forms)         | Panel       | Allows inserting and updating application data, as well as modifying configuration directly from your Grafana dashboard.        |
| [Business Input](business-input)         | Data source | Allows storing and emulating your data using JavaScript and LLM.                                                                |
| [Business Media](business-media)         | Panel       | Allows to create the desired visual experience by mixing and matching any supported formats for multimedia file visualizations. |
| [Business News](business-news)           | Data source | Retrieves RSS and Atom feeds and allows visualizing them using Business Text and other panels.                                  |
| [Business Satellite](business-satellite) | Data source | Allows to retrieve data from local and remote Grafana instances via HTTP API.                                                   |
| [Business Table](business-table)         | Panel       | Allows to simplify data visualization in table format.                                                                          |
| [Business Text](business-text)           | Panel       | Allows to convert plain text and table data into visually appealing, easy-to-read information cards.                            |
| [Business Variable](business-variable)   | Panel       | Allows you to have dashboard filters in a separate panel to place anywhere on the dashboard.                                    |
