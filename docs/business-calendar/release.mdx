## 3.9.1 (2025-03-18)

This release focuses on improving the user experience with an updated "no events" message, a more stable Yearly view layout for smaller panel sizes, and the ability to display events from previous years.

## 3.9.0 (2025-02-20)

This version enhances testing with improved end-to-end (e2e) tests, updates to Grafana 11.5 with the latest dependencies, and adds attestation to the release workflow for better security.

## 3.8.0 (2024-10-25)

Introduces Multi-Day Events in Yearly View, improves migration for thresholds color filters, enhances compatibility with Scenes dashboards, and updates to Grafana 11.3 with the latest dependencies.

<BlogPost id="calendar-panel-3.8.0" />

## 3.7.0 (2024-09-15)

Enhances the Yearly View to display events and updates to Grafana 11.2 with the latest dependencies for improved performance and compatibility.

## 3.6.0 (2024-08-19)

Refines user interactions with an improved tooltip to prevent unintended drawer openings, better date-time formatting for multiple languages, and hides event actions when none are available.

## 3.5.0 (2024-07-25)

Adds event details tooltip on hover, introduces an option to disable event time display, updates daily view on time range changes, aligns descriptions with field order, and adjusts dates to use the timeZone option.

<BlogPost id="calendar-panel-3.5.0-20240815" />

## 3.4.0 (2024-07-16)

Optimizes testing workflows with Docker, introduces font size configuration for better customization, and updates to Grafana 11.1 dependencies for compatibility.

## 3.3.0 (2024-06-19)

Improves variable handling with static datasource for time range, supports multiple description fields, adds threshold-based color options, and includes Portuguese language support for Grafana 11.

## 3.2.0 (2024-06-11)

Enhances usability with updated tutorial videos, formatted event descriptions, a revamped Details drawer layout, support for additional Time Range sources, refined event formats, and improved Agenda view features, alongside dependency updates and Portuguese localization.

## 3.1.0 (2024-05-19)

Introduces Agenda (List) View, customizable Event Bubbles, improved multi-day windows, and updates to Grafana 11 for better compatibility and performance.

## 3.0.0 (2024-05-02)

:::warning Breaking Changes

- Rebranded to Business Calendar panel
- Replaced Legacy Calendar with Big Calendar
- Requires Grafana 10 or 11
  :::

Marks a major update with rebranding to Business Calendar panel, a transition to Big Calendar, preparation for Grafana 11, and the addition of plugin e2e tests while preserving dashboard refresh on time range changes.
