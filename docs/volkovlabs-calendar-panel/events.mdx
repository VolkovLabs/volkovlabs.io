# Events

The Calendar visualization panel allows displaying events from any data source.

## Fields

To visualize data select fields in the panel's options.

| Field       | Description                                         |
| ----------- | --------------------------------------------------- |
| Text        | Event text. Defaults to the first text field.       |
| Description | Event description.                                  |
| Start time  | Event start time. Defaults to the first-time field. |
| End time    | Event end time.                                     |
| Labels      | Event labels.                                       |
| Color       | Event color.                                        |

## Duration

- If the event has only the Start time field configured, events are considered instantaneous.
- If the event has Start and End time fields configured, events are considered to have a duration.
- Any events that have End time configuration and equal null are considered ongoing.

## Color

The background color of the event uses the threshold configuration for the selected field.
