# Big Calendar

:::note Legacy Calendar

Legacy Calendar was replaced by Big Calendar in version 3.0.0.
:::

Big Calendar is an upgraded implementation of the calendar component built based on the powerful React library.

## Toolbar

It introduces a multi-language toolbar that allows you to:

1. Switch between the Year, Month, Week, Work Week, and Day views.
2. Switch back to today's events.
3. Display events from the previous and subsequent time ranges.
