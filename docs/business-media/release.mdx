## 6.0.0 (2024-07-29)

:::warning Breaking Changes

- Image links in panel options must now use Data links
- PDF fields require updating the media source type to PDF
  :::

A major update with breaking changes for image links and PDF fields, alongside significant enhancements like video poster support and mixed media slideshows, updated for Grafana 11.1.
