# Baidu Maps

:::info

Supported since Apache ECharts visualization panel 3.6.0.

:::

Baidu Maps are loaded using API v3 and require Access Key.

You can get the access key from [https://lbsyun.baidu.com/apiconsole/key#/home](https://lbsyun.baidu.com/apiconsole/key#/home).

## Features

- Loading Baidu Maps takes 2-3 seconds.
- The callback function `bmapReady` will be executed on load. The name can be changed in the Panel options.
- While loading, animation can be displayed using the code below.
