# Google Maps

:::info

Supported since Apache ECharts visualization panel 3.8.0.

:::

Google Maps are loaded using Maps API and require Access Key.

## Access Key

You can get the access key from [https://console.cloud.google.com/apis/credentials](https://console.cloud.google.com/apis/credentials).

## Demo

Try the panel with [ECharts Examples dashboard in edit mode](https://echarts.volkovlabs.io/d/X1mkMIFVz/geo-map?orgId=1&editPanel=13).

## Features

- Loading Google Maps takes ~1-2 seconds.
- The callback function `gmapReady` will be executed on load. The name can be changed in the Panel options.
