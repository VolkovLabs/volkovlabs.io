# Visual Editor


:::info
This feature is in the beta version. As of now, it should work for simple charts. If you need anything advanced, please, use the **Code** mode.
:::

<Image
  title="The Visual Editor is a new and exciting feature added to simplify the option settings for the charts."
  src="/img/blog/2024-04-06-echarts-panel-6.0.0/vis-editor.png"
/>

This is one of the most exciting features. Instead of using JavaScript code, you can use UI elements to specify many chart parameters. 

### Visual Editor Examples

To get you started faster, we have prepared many examples of how the **Visual Editor** can be used. 

Please, check out the [Examples section](/plugins/volkovlabs-echarts-panel/examples/) of the documentation.
