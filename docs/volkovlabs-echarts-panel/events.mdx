# Event Handling

Users can trigger related events through their operation. React on Mouse and other events using `echartsInstance`.

For more examples and details, take a look at the ECharts documentation [Event and Action](https://apache.github.io/echarts-handbook/en/concepts/event/).

## Mouse Click

```js
/**
 * On Mouse Click
 */
echartsInstance.on("click", (params) => {
  notifySuccess(["Event", "On Click"]);
  echartsInstance.resize(); // to redraw visualization
});
```

```js
/**
 * On Double Click
 */
echartsInstance.on("dblclick", (params) => {
  echartsInstance.resize();
});
```

## Update Time Range

The data zoom feature allows selecting the time range on the chart. When the time range is selected the event can trigger updating the dashboard's time range using the `locationService` parameter similar to Grafana native functionality.
