## How to use

1. Open the [echarts.volkovlabs.io](https://echarts.volkovlabs.io) with examples.
2. Find the chart that looks closest to what you need to build.
3. Open the chart of interest in the 'Edit' mode.

   - Copy the code from the right-hand side
   - Start experimenting right away. You can't save the changes.

4. Paste the copied code into your Apache ECharts visualization panel on the Grafana dashboard.
5. You should see the working chart right away.

   - You might need to refresh the chart to update the state.
