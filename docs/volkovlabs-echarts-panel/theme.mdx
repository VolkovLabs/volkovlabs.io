# Theme Editor

:::info Version

Theme Editor is supported by the Apache ECharts Panel starting from version 4.4.0.

:::

Theme Editor allows using custom themes generated with [Theme Builder](https://echarts.apache.org/en/theme-builder.html).

<Image
  title="Apache ECharts Panel supports custom themes from Theme Builder."
  src="/img/plugins/volkovlabs-echarts-panel/theme.png"
/>

## Theme Builder

The Theme Builder created by the Apache ECharts team provides advanced options to create fully customizable themes. A theme configuration is saved into a JSON file, which you can download and paste into Theme Editor.

<Image
  title="Theme Builder allows creating custom themes for Apache ECharts Panel."
  src="/img/plugins/volkovlabs-echarts-panel/theme-builder.png"
