# Theme Editor

:::info Version

The Business Charts Panel supports Theme Editor since version 4.4.0.

:::

Theme Editor allows using custom themes generated with [Theme Builder](https://echarts.apache.org/en/theme-builder.html).

<Image
  title="The Business Charts Panel supports custom themes from Theme Builder."
  src="/img/plugins/volkovlabs-echarts-panel/theme.png"
/>

## Theme Builder

Theme Builder allows you to create fully customizable themes. Configuration is saved into the JSON file, which can be downloaded and inserted into Theme Editor.

<Image
  title="Theme Builder allows creating custom themes for the Business Charts Panel."
  src="/img/plugins/volkovlabs-echarts-panel/theme-builder.png"
/>
