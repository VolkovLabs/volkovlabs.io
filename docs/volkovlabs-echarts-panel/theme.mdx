# Theme Editor

:::info Version

Theme Editor is supported by the Apache ECharts visualization panel starting from version 4.4.0.

:::

Theme Editor allows using custom themes generated using the [Theme Builder](https://echarts.apache.org/en/theme-builder.html).

<Image
  title="Apache ECharts visualization panel supports custom theme from Theme Builder."
  src="/img/plugins/volkovlabs-echarts-panel/theme.png"
/>

## Theme Builder

The Theme Builder created by the Apache ECharts team provides many options to create fully customizable themes. Configuration is saved in the JSON file, which can be downloaded and pasted into the Theme Editor.

<Image
  title="Theme Builder allows creating a custom theme for Apache ECharts."
  src="/img/plugins/volkovlabs-echarts-panel/theme-builder.png"
/>
