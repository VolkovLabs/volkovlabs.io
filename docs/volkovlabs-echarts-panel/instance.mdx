# ECharts Instance

The `echartsInstance` is the created instance (container) of the Apache ECharts. It allows to

- Get the width and height of the ECharts container.
- Get the DOM (Document Object Model) to include additional elements.
- Resize.
- Handle Events.
- Update options, etc.

The [echartsInstance](https://echarts.apache.org/en/api.html#echartsInstance) described in the ECharts documentation.

## Scale when resizing

To scale the content when the panel is resized using the instance methods to retrieve the width and height of the panel.

```javascript
  graphic: {
    type: "image",
    style: {
      image: `data:image/svg+xml;utf8,${SVG}`,
      width: echartsInstance.getWidth(),
      height: echartsInstance.getHeight(),
    },
  },
```
