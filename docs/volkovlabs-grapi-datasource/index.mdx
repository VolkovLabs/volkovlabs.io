# Business Satellite

:::info announcement
The HTTP API data source joins the [Business Suite](/plugins/) for Grafana and with that gets an elevated name. Now, the same amazing open source plugin is called the Business Satellite data source.

<Image
  title="The HTTP API data source now is called the Business Satellite data source."
  src="/img/plugins/volkovlabs-grapi-datasource/new-name.png"
  width="60%"
/>
:::

The Business Satellite Data Source for Grafana allows retrieving data from local and remote Grafana instances via HTTP API.

<Youtube
  id="0zibOEGqTJ8"
  title="Easy access to Grafana backend using Business Satellite data source."
/>

## Requirements

- **Grafana 9** and **Grafana 10** are required for major version 2.
- **Grafana 8.5** and **Grafana 9+** are required for major version 1.

<GettingStarted />

## Highlights

- Connects to Local and Remote Grafana instances via HTTP API using API Keys and Tokens.
- Allows getting Health information.
- Allows retrieving Annotations, Alerts, and Data Sources.
