# Toolbar

The Base64 Image/Video/Audio/PDF visualization panel provides a toolbar for images and PDF files.

:::info Version

Toolbar for images supported in the Base64 panel starting from version 3.6.0.

Pan and Pinch zoom are supported in the Base64 panel starting from version 3.7.0.
:::

## PDF

The toolbar for PDF files is provided out of the box in most browsers and can be disabled using panel options if supported.

<Image
  title="Base64 panel provides a toolbar for PDF files."
  src="/img/plugins/volkovlabs-image-panel/pdf.png"
/>

## Images

The toolbar for images allows to:

- Navigate (previous, next) thru returned images.
- Download the selected image.
- Zoom in Full Screen and Pan and Pinch modes.
