## 3.5.0 (2025-04-03)

This release introduces new filtering capabilities for dashboard search and annotations, alongside an upgrade to Grafana 11.5 with updated dependencies for improved compatibility and performance.

## 3.4.0 (2024-12-25)

Focused on enhancing end-to-end testing and ensuring compatibility with Grafana 11.4 through updated dependencies.

## 3.3.0 (2024-11-17)

A maintenance release aligning the plugin with Grafana 11.3 and incorporating the latest dependency updates.

## 3.2.0 (2024-07-29)

:::warning Breaking Changes
Requires Grafana 10.2 or 11
:::

Introduces support for variables in Dashboard Scenes and Grafana alerts, with a breaking change requiring Grafana 10.2 or 11.

## 3.1.0 (2024-07-16)

Enhances functionality with dashboards metadata requests and health checks, setting a minimum Grafana version of 10.1.0.

## 3.0.0 (2024-07-11)

:::warning Breaking Changes

Requires Grafana 10.1 or 11
:::

A major update preparing for Grafana 11 compatibility, optimizing testing with Docker, and introducing breaking changes requiring Grafana 10.1 or 11.
