# Imports

Static data source supports importing in code editor mode and using import libraries.

### Mermaid

Mermaid is a popular JavaScript-based diagramming and charting tool that dynamically creates and modifies diagrams using Markdown-defined text definitions.

Example of usage Mermaid in Business Text

Use the following for the **Content**

```js
<pre class="mermaid">
    graph LR
    A --- B
    B-->C[fa:fa-ban {{data.0.test}}]
    B-->D(fa:fa-spinner);
</pre>
```

Use the following for the **JavaScript->Before Content Rendering**

```js
import("https://esm.sh/mermaid").then(({ default: mermaid }) => {
  mermaid.initialize({ startOnLoad: true });

  mermaid.run({
    querySelector: ".mermaid",
    suppressErrors: false,
  });
});
```

Code to copy for `Business Text` panel example

Use the following for the **Content**

```html
<pre class="mermaid">
    graph LR
    A --- B
    B-->C[fa:fa-ban {{data.0.test}}]

```

Use the following for the **Code editor**

```js
import("https://esm.sh/mermaid").then(({ default: mermaid }) => {
  mermaid.initialize({ startOnLoad: true });

  mermaid.run({
    querySelector: ".mermaid",
    suppressErrors: false,
  });
});

const array = Array.from({ length: 3 }, (v, i) => `${i + 1}`);

const result = {
  ...frame,

return Promise.resolve(result);
```

:::info Note
JavaScript Values Editor must return data frame
:::

### Chart.js

Code to copy for `Business Text` panel example

Use the following for the **Content**

```html
<canvas id="myChart" />
```

Use the following for the **Code editor**

```js
import("https://esm.sh/chart.js").then(({ Chart, registerables }) => {
  Chart.register(...registerables);

  /**
   * Cleanup
   */
  if (this.chartInstance) {
    this.chartInstance.destroy();
  }

  const ctx = document.getElementById("myChart");

  this.chartInstance = new Chart(ctx, {
    type: "bar",
    data: {
      labels: ["Red", "Blue", "Yellow", "Green", "Purple", "Orange"],
      datasets: [
        {
          label: "# of Votes",
          data: [12, 19, 3, 5, 2, 3],
          borderWidth: 1,
        },
      ],
    },
    options: {
      scales: {
        y: {
          beginAtZero: true,
        },
      },
    },
  });
});

const array = Array.from({ length: 3 }, (v, i) => `${i + 1}`);

const result = {
  ...frame,
  fields: frame.fields.map((field) => {
    return {
      ...field,
      values: array.map((item) => {
        return item;
      }),
    };
  }),
};

return Promise.resolve(result);
```
