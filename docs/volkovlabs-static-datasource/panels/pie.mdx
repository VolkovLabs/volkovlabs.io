# Pie Chart

You can use the Static data source to test category panels like Pie Charts.

## Fields

- String field `name` for the category name.
- Number field `amount` for numeric value.

```javascript
const values = [
  ["Search Engine", "Direct", "Email", "Union Ads", "Video Ads"],
  [1048, 735, 580, 484, 300],
];

const result = {
  ...frame,
  fields: frame.fields.map((field, index) => ({
    ...field,
    values: values[index],
  })),
};

return Promise.resolve(result);
```
