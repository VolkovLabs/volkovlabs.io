# Units

:::info Version

Dynamic Text Panel supports override data values via uni
:::

In the JavaScript->before rendering content option a user can register custom handlebars.
Example with [custom handlebars Helper](../code/#custom-handlebars-helper)

Use `context.handlebars.registerHelper` and `display` method from data field in code editor to transform value, and display transformed value.
As example transform value from `1048576` to `1.05 MB`
