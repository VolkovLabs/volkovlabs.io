# Styles

The Styles editor allows adding new CSS styles and overrides the default using the dedicated code editor with CSS syntax highlight.

Dashboard variables are replaced automatically in the CSS styles.

:::info Version

Supported since Dynamic Text panel 3.0.0.

Dashboard variables are supported in CSS styles since 4.0.0.

:::

## Add new styles

You can define your styles and use them in the template.

### Styles

```css
td.name {
  border: 0;
  background-color: #5d3fc4;
  color: white;
}

b.name {
  font-family: silom;
  font-size: 20px;
}
```

### Template

```html
<td class="name">
  <p><b class="name">{{first_name}} {{last_name}}</b></p>
</td>
```

### Padding and margins

To display content without padding and margins override the parent CSS style.

```css
& {
  padding: 0;
  margin: 0;
}
```

## Code Syntax Highlight

We included the Accessibility (a11y) syntax highlight style for light and dark themes. You can override it in the Styles editor by copying one of the styles from the [project source files](https://github.com/highlightjs/highlight.js/tree/main/src/styles).
