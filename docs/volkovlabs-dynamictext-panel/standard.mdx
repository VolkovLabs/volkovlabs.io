# Standard Options

:::info Version

The Business Text Panel supports the **Standard options** and **Overrides** starting from version 5.0.0.
:::

The Business Text Panel supports the following **Standard options**:

- **Unit**,
- **Decimals**.

<Image
  title="The Business Text supports Units and Decimals defined in the Standard options and Overrides."
  src="/img/plugins/volkovlabs-dynamictext-panel/features/units/panel.png"
/>

## Overrides

The **Unit** and **Decimals** can be set for a specific field or data frame using **Overrides**.

  title="Override unit for fields with name Value."
