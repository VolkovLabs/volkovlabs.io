## 1.3.0 (2025-05-28)

This release introduces several enhancements to the editor component and overall user experience. Key additions include icons for visual feedback, horizontal and vertical layout options, Time-Picker input support, HTML delimiters, current time picker highlighting, button row dropdowns, and a panel freeze feature for scrolling dashboards. Additionally, end-to-end (E2E) testing has been implemented for reliability. The update also includes an upgrade to Grafana 12.0 with refreshed dependencies for better compatibility and performance.
