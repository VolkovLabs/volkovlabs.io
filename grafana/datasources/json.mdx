# JSON API Data Sources

API data source in Grafana is a plugin that downloads a file via a provided link and then, following the given instructions reads the file and converts it into a Grafana data frame.

<Image
  title="API Data Source uses GET/POST request to retrieve data from external systems."
  src="/img/grafana/api-concept.png"
/>

## Popular data sources

The most popular API data sources are

- [JSON API](https://grafana.com/grafana/plugins/marcusolsson-json-datasource/)
- [Infinity](https://grafana.com/grafana/plugins/yesoreyeram-infinity-datasource/)
- [SimPod](https://grafana.com/grafana/plugins/simpod-json-datasource/)
