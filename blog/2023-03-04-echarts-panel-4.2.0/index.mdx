# Apache ECharts Panel 4.2.0

We released a new version of the Apache ECharts Panel 4.2.0 for Grafana. This maintenace release

- Added `EventBus` parameter to publish events.
- Removed panel padding allowing to use of panel's space corner to corner.
- Updated default background to `transparent` for Light and Dark themes.
- Updated to the latest Grafana 9.4.3 toolkit and workflows.

:::info
The plugin was submitted to the Grafana team for review.
:::

<!--truncate-->

The data zoom feature allows selecting the time range on the chart. When the time range is selected the event can trigger updating the dashboard's time range using the `locationService` parameter similar to Grafana behavior.

## Getting Started

Apache ECharts visualization panel can be installed from the [Grafana Catalog](https://grafana.com/grafana/plugins/volkovlabs-echarts-panel/) or utilizing the Grafana command line tool.

For the latter, use the following command.

```bash
grafana-cli plugins install volkovlabs-echarts-panel
```
