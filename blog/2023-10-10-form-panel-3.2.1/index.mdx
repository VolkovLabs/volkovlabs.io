# Data Manipulation Panel 3.2.1

We are happy to announce the release of Data Manipulation Panel 3.2.1. This major release includes the following updates:

- Added min and max date for date time element.
- Added mapping Data Source values to elements.
- Updated element Show If to support variables.
- Added Query Field Picker for Initial Request.
- Added File element type for File Upload.
- Added loading states for Initial, Update and Reset button actions.
- Added icons for radio and select options.

:::info Grafana Catalog
The plugin was updated in the Grafana Plugins catalog on September 11, 2023.
:::

## Getting Started

You can install Data Manipulation Panel from the [Grafana Plugins catalog](https://grafana.com/grafana/plugins/volkovlabs-form-panel/) or using the Grafana command line tool.

For the latter, please use the following command:

```bash
grafana-cli plugins install volkovlabs-form-panel
```
