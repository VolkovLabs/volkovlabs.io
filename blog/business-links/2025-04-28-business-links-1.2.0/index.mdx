# Business Links 1.2.0

We are excited to announce the release of the Business Links panel 1.2.0. This feature release includes the following updates:

- Added Option to specify menu position
- Signed as Community plugin
- Upgraded to Grafana 11.6 with dependency updates
- Added Option to sort tabs and prioritize selected tabs first
- Added Option to highlight the current dashboard or link
- Added Option to display dropdown menus on hover instead of click
- Initial release of the Business Links panel based on Volkov Labs templates
