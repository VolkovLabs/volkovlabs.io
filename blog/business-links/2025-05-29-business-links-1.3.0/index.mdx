# Business Links 1.3.0: Enhanced Features and Grafana 12.0 Compatibility

Weâ€™re excited to announce **Business Links 1.3.0**, the latest update to our powerful navigation panel for Grafana. Built on top of Grafana's native link functionality, Business Links offers unmatched flexibility and customization, making it easier than ever to access dashboards and external resources.

## What's New in Business Links 1.3.0?

This release introduces exciting features, usability enhancements, and full compatibility with Grafana 12.0. Here's what you can expect:

- **Editor Icons**: Visual icons for intuitive navigation.
- **Layout Options**: Choose between horizontal and vertical grid layouts for a tailored look.
- **Time-Picker Input**: Seamlessly interact with time-based data.
- **HTML Type**: Format content with improved structure.
- **Time Picker Highlight**: Easily spot the active time picker.
- **Button Row for Dropdowns**: Enhance interaction with dropdown menus.
- **Freeze on Scroll**: Keep the panel visible on long dashboards.
- **End-to-End Testing**: Ensure reliability with rigorous testing.
- **Grafana 12.0 Compatibility**: Stay up-to-date with the latest Grafana version.

Dive into the details below to learn more about each feature!

### Editor Icons for Intuitive Navigation

We've added visual icons, making it easier to identify the meaning of each element. This small but impactful update enhances usability, especially for new users.

Enhance the user experience by utilizing the extensive list of predefined icons or maintain the specific style with the **Use custom icon** option.

### Horizontal and Vertical Layouts

Customize the look of your Business Links panel with new horizontal and vertical grid layout options. Whether you prefer a compact or expansive design, these layouts adapt to your dashboard needs.

### Time-Picker Input for Seamless Interactions

Interact with time-specific data directly from the Business Links panel using the new Time-Picker input type. This feature simplifies time-based tasks, saving you valuable clicks.

### HTML Type for Creative Formatting

Organize your content with the new HTML element type. This addition ensures a polished and structured presentation of links and resources within the panel.

You can go as creative as needed using HTML capabilities.

### Time Picker Highlight for Better Visibility

Never lose track of your active time picker! The new highlight option makes it stand out, improving visibility and user experience.

### Button Row for Dropdown Menus

Dropdown menus are now more interactive with the addition of a button row type. This update makes navigation smoother and more accessible.

### Freeze on Scroll for Long Dashboards

Navigating long dashboards? The new "Freeze on Scroll" feature keeps the Business Links panel fixed in place, ensuring constant access no matter where you are on the page.

### End-to-End Testing for Reliability

We've implemented comprehensive end-to-end (E2E) testing to guarantee the stability and reliability of Business Links. Rest assured, this update is built to perform.

### Full Compatibility with Grafana 12.0

Business Links 1.3.0 is fully compatible with Grafana 12.0. With updated dependencies, you can enjoy seamless integration and enhanced performance on the latest Grafana version.
