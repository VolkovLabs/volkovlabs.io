# Business Links 1.3.0: Enhanced Features and Grafana 12.0 Compatibility

Weâ€™re thrilled to unveil **Business Links 1.3.0**, the latest update to our powerful navigation panel for Grafana. The Business Links panel builds on top of the regular Grafana links functionality, making it more flexible and customizable. It is designed to streamline access to dashboards and external resources.

## What's New in Business Links 1.3.0?

This release brings a host of new features, usability improvements, and compatibility updates to elevate your Grafana experience. Here's a quick overview of the key enhancements:

- **Editor Icons**: Added visual icons in the editor for better feedback and ease of use.
- **Layout Options**: Introduced horizontal and vertical layouts for greater customization.
- **Time-Picker Input**: Added support for time-based interactions with a new input type.
- **HTML Delimiter**: Implemented a new delimiter type for improved content formatting.
- **Time Picker Highlight**: Added an option to highlight the current time picker for better visibility.
- **Button Row for Dropdowns**: Enhanced user interaction with a new button row type in dropdown menus.
- **Freeze on Scroll**: Added a feature to keep the panel fixed while scrolling through long dashboards.
- **End-to-End Testing**: Implemented E2E testing to ensure reliability and stability.
- **Grafana 12.0 Compatibility**: Upgraded dependencies for seamless integration with Grafana 12.0.

### Editor Icons for Enhanced Visual Feedback

We've added visual icons to the editor component to provide better feedback and improve ease of use. This update makes the experience more intuitive, helping users quickly identify tools and options within the Business Links panel.

### Horizontal and Vertical Layout Options

To offer greater customization, Business Links 1.3.0 introduces new horizontal and vertical layout options. These layouts allow users to tailor the panel's appearance to better fit their dashboard design and workflow preferences.

### Time-Picker Input for Time-Based Interactions

A new Time-Picker input type has been added to support time-based interactions. This feature enables users to interact with time-specific data or settings directly from the Business Links panel, streamlining time-related tasks.

### HTML Delimiter for Improved Content Formatting

We've implemented a new HTML delimiter type to enhance content formatting and separation. This addition allows for cleaner and more structured presentation of links and resources within the panel.

### Time Picker Highlight for Better Visibility

To improve user experience, an option to highlight the current time picker has been added. This feature ensures that the active time picker stands out, making it easier to spot at a glance.

### Button Row for Dropdown Menus

A new button row type for dropdown menus has been introduced to enhance user interaction and accessibility. This update provides a more interactive and user-friendly way to navigate options within dropdowns.

### Freeze on Scroll for Long Dashboards

For better usability in long dashboards, we've added a feature to freeze the Business Links panel on scroll. This ensures that the panel remains accessible at all times, even when navigating through extensive content.

### End-to-End Testing for Reliability

To ensure the stability and reliability of Business Links, we've implemented end-to-end (E2E) testing. This rigorous testing process helps us deliver a robust and dependable experience for all users.

### Compatibility with Grafana 12.0

Business Links 1.3.0 has been upgraded to support Grafana 12.0, with updated dependencies to ensure seamless integration and improved performance. This update guarantees that the panel works flawlessly with the latest Grafana version.
