# Calendar Panel 1.4.0 supports Grafana 10

A new version of the Calendar panel 1.4.0 was released today. This maintenance release

- Tested with Grafana 10 Preview.
- Added Text Formatting, Mappings and Overrides.
- Removed unused Standard Options.
- Increase event spacing to prevent overflow.
- Increased Test coverage and added end-to-end test workflow.
- Migrated to Plugin Tools 1.5.2 with Node 18 and npm.

:::info Grafana Catalog
The plugin was submitted to the Grafana team for review.
:::

## Getting Started

The calendar panel can be installed from the [Grafana Catalog](https://grafana.com/grafana/plugins/marcusolsson-calendar-panel/) or utilizing the Grafana command line tool.

For the latter, use the following command.

```bash
grafana-cli plugins install marcusolsson-calendar-panel
```
