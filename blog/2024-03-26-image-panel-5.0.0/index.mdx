# Base64 Image/Video/Audio/PDF Panel 5.0.0

We are excited to announce the release of the Base64 Image/Video/Audio/PDF Panel 5.0.0. This major release includes the following updates:

- Requires Grafana 10 and Grafana 11.
- Added compatibility with Grafana 10.4.1.
- Added the ability to customize the alert message.
- Added infinity play option.

:::info Grafana Plugins catalog
The plugin is under review.
:::

## Getting Started

You can install Base64 Image/Video/Audio/PDF Panel from the [Grafana Plugins catalog](https://grafana.com/grafana/plugins/volkovlabs-image-panel/) or using the Grafana command line tool.

For the latter, please use the following command:

```bash
grafana-cli plugins install volkovlabs-image-panel
```

## YouTube Tutorial
