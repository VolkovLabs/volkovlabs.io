# Yearly view, Internationalization and Grafana 11.3 Scenes updates in Business Calendar 3.8.0

We are happy to announce the release of the Business Calendar panel 3.8.0. This release includes the following updates.

- Added display single and multi day events in Yearly View.
- Updated Date time format for different languages.
- Added compatibility with Grafana 11.3 and Scenes dashboards.
