# Business Calendar Panel: A Journey of Innovation and Enhancement

The **Business Calendar panel** has evolved significantly since its initial release, transforming into a powerful tool for visualizing events and schedules within Grafana. With each update, we've introduced new features, improved user experience, and ensured compatibility with the latest Grafana versions.

Below is a comprehensive summary of the innovations and enhancements across all versions of the Business Calendar panel, reflecting our commitment to delivering a robust and user-friendly plugin.

## Early Foundations and Initial Releases (1.0.0 - 1.4.0)

The journey began with the **1.0.0 release** under Volkov Labs maintenance, establishing the panel with compatibility for Grafana 9.1.6. Subsequent updates introduced critical features and refinements:

- **1.1.0 (2022-12-12)**: Added support for annotations, enhanced documentation, and included a YouTube tutorial for JSON API events, alongside compatibility with Grafana 9.3.1.
- **1.2.0 (2023-02-23)**: Introduced color selection for calendar entries, refactored plugin options with field pickers, and aligned the week start day with Grafana system settings.
- **1.3.0 (2023-03-08)**: Improved the Drawer component with split labels and annotation tags, supported event coloring by ID, and added color schemes.
- **1.4.0 (2023-06-10)**: Enhanced text formatting, increased test coverage, and integrated E2E Cypress testing while ensuring compatibility with Grafana 9.5.2.

:::info Early Releases
These early releases laid a strong foundation, focusing on usability, customization, and integration with Grafana's ecosystem.
:::

## Major Overhauls and Big Calendar Introduction (2.0.0 - 2.5.0)

The **2.0.0 release** marked a significant milestone with the introduction of the **Big Calendar component**, a major UI overhaul, and compatibility with Grafana 9.2 and 10.0. Key innovations included:

- Quick links, navigation buttons, and calendar resizing for improved usability.
- Increased test coverage for reliability.

Subsequent updates built on this foundation:

- **2.1.0 (2023-08-14)**: Added annotation types, display limits, and language messages for Big Calendar.
- **2.2.0 (2023-10-06)**: Introduced Event Location display, improved text color contrast, and added new view options (day, week, work week, month).
- **2.3.0 (2023-11-01)**: Debuted the **Yearly View**, added internationalization (Spanish, French, German, Chinese), and enabled scroll-to-time functionality.
- **2.4.0 (2024-03-07)**: Enhanced Big Calendar to version 1.10.3, added date format selection, and introduced an option to hide end time.
- **2.5.0 (2024-03-20)**: Updated event details tag styles and fixed location display issues in various views.

:::info Expanding
This era focused on expanding visualization options and enhancing the user interface with the powerful Big Calendar framework.
:::

## Rebranding and Modernization (3.0.0 - 3.9.1)

The **3.0.0 release (2024-05-02)** was a transformative update, rebranding the plugin as the **Business Calendar panel** and replacing the Legacy Calendar with Big Calendar. Key changes included:

- Compatibility with Grafana 10 and 11.
- Introduction of plugin end-to-end (e2e) tests for quality assurance.

Further innovations followed:

- **3.1.0 (2024-05-19)**: Introduced **Agenda (List) View**, customizable Event Bubbles, and improved multi-day windows.
- **3.2.0 (2024-06-11)**: Enhanced usability with formatted event descriptions, a revamped Details drawer layout, and support for additional Time Range sources.
- **3.3.0 (2024-06-19)**: Improved variable handling, supported multiple description fields, and added threshold-based color options.
- **3.4.0 (2024-07-16)**: Optimized testing workflows with Docker and introduced font size configuration.
- **3.5.0 (2024-07-25)**: Added event details tooltip on hover, an option to disable event time display, and better date-time formatting for multiple languages.
- **3.6.0 (2024-08-19)**: Refined user interactions with improved tooltips and hid event actions when unavailable.
- **3.7.0 (2024-09-15)**: Enhanced Yearly View to display events more effectively.
- **3.8.0 (2024-10-25)**: Introduced **Multi-Day Events in Yearly View**, improved migration for thresholds color filters, and enhanced compatibility with Scenes dashboards.
- **3.9.0 (2025-02-20)**: Strengthened security with attestation in the release workflow and updated to Grafana 11.5.
- **3.9.1 (2025-03-18)**: Improved user experience with an updated "no events" message, a stable Yearly View layout for smaller panels, and support for displaying events from previous years.

:::info Rebranding
This phase emphasized modern features, localization (including Portuguese support), and seamless integration with the latest Grafana versions.
:::

## Key Themes of Innovation

Across all releases, the Business Calendar panel has focused on several core areas of innovation:

- **Visualization Enhancements**: From the introduction of Big Calendar to new views like Yearly and Agenda, we've continuously expanded how users can visualize events.
- **User Experience**: Features like tooltips, customizable event bubbles, font size options, and improved layouts ensure intuitive interaction.
- **Customization**: Options for color schemes, date formats, and event display settings empower users to tailor the panel to their needs.
- **Compatibility and Performance**: Regular updates to support the latest Grafana versions (up to 11.5) and dependency optimizations keep the panel robust and future-proof.
- **Testing and Security**: Increased test coverage, E2E testing, and workflow attestation reflect our dedication to reliability and trust.

## Looking Ahead

The Business Calendar panel continues to evolve, driven by community feedback and the need for innovative visualization tools in Grafana. Whether you're managing schedules, tracking events, or integrating with complex dashboards, our panel is designed to meet your needs with flexibility and style.

Stay tuned for future updates, and thank you for being part of this journey! For detailed release notes and specific version information, check out our [official documentation](/plugins/business-calendar/).
