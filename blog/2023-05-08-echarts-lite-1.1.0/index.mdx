# Apache ECharts Lite Panel 1.1.0

We released a new version of the Apache ECharts Lite Panel 1.1.0 for Grafana. This maintenance release

- Updated to the latest Grafana 9.5.1 toolkit.
- Updated plugin Id `volkovlabs-echarts-lite-panel`.
- Based on Apache ECharts 5.4.2.

:::info Fork

The Apache ECharts Lite plugin is a fork of the Apache ECharts panel that provides only Core functionality without maps and extensions. It's designed to use in Application plugins and IoT projects.

The Apache ECharts panel is a Community plugin available in the Grafana Catalog.
:::

## Compare to Apache ECharts panel

| Features    | Apache ECharts panel | Apache ECharts Lite panel |
| ----------- | -------------------- | ------------------------- |
| 3D          | Yes                  | No                        |
| Liquid Fill | Yes                  | No                        |
| Maps        | Yes                  | No                        |
| Statistics  | Yes                  | No                        |
| Module Size | 2 megabytes          | 1 megabyte                |
| Signed      | Community            | Private                   |
