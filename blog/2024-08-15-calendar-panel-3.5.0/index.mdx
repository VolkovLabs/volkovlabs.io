# Business Calendar 3.5.0

We are happy to announce the release of the Business Calendar panel 3.5.0. This release includes the following updates.

- Added event details in tooltip.
- Added option to disable event time for month.
- Added Font size configuration.
- Added multiple description fields and format.
- Added Portuguese language in Grafana 11.
- Updated Details drawer layout.
- Added Agenda (List) View.

:::info Grafana Plugins catalog
The plugin was updated in Grafana Catalog on August 1, 2024.
:::
