# Business Media 3.4.0

We are excited to announce the release of the Business Media Panel 3.4.0. This maintenance release includes the following updates:

- Added compatibility with Grafana 9.3.2.
- Updated the README field and [plugin documentation](/plugins/business-media) to explain how to use the panel with data sources and supported formats.
- "New Year" edition.
