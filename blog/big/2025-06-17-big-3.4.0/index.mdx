We’re excited to unveil **Business Intelligence 3.4.0**, a transformative update to the [Business Intelligence platform](/big/). This release brings powerful new features and enhancements to both **Business Studio** and **Business Engine**, designed to streamline collaboration, supercharge alerting, and boost performance. Whether you're a data analyst, DevOps engineer, or business leader, this update will take your Grafana workflows to the next level.

## What’s New in Business Intelligence 3.4.0?

This release is packed with innovations to enhance user experience and integration. Here’s a quick overview of the standout features:

- **Panel Screenshots for Alerting**: Instantly view panel screenshots during alerting events for faster data insights.
- **Renderer Microservice**: A new service to generate panel previews, setting the stage for advanced reporting features.
- **Alerting Event Details**: Access a dedicated page for in-depth notification details on alerting events.
- **Prometheus Metrics Update**: Monitor microservices with exposed metrics via Server API on port 3001.
- **Google Login Integration**: Seamlessly connect to Google Firebase for easy access to shared organization engines, fostering collaboration.
- **Enhanced Engine Management**: Enjoy improved sorting for engine lists and support for shared organization engines.

Ready to revolutionize your Grafana workflows? Dive into Business Intelligence 3.4.0 today!

## Release Notes

### Business Studio 3.4.0

- **Google Login Integration**: Connect effortlessly to Google Firebase for access to organization engines and enhanced teamwork.
- **Panel Screenshots for Alerting**: Get quick visual previews of panels during alerting events.
- **Enhanced Engine Management**: Benefit from updated sorting and support for shared organization engines.
- **Improved UI Experience**: Enjoy smoother interactions with added animations and a refined refresh mechanism.
- **Alerting Event Details**: Explore detailed notification information on a dedicated page for alerting events.

### Business Engine 3.4.0

- **Google Firebase Authentication**: Securely authenticate users and manage shared engine configurations.
- **Renderer Microservice**: Generate panel previews with a new service, paving the way for future reporting tools.
- **Prometheus Metrics Update**: Access microservice metrics via Server API on port 3001 for better monitoring.
- **Testing Improvements**: Enhanced E2E and unit tests for alert rules and actions ensure rock-solid reliability.
