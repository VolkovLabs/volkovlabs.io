# Business alerting, action history, and event page updates in Business Intelligence 2.6.0

We are excited to announce the public preview release of the Business Intelligence platform 2.6.0. It includes the following updates:

- Integrated an alerting page to present events across high availability clusters of business engines.
- Added action history to track messages that have been sent.
- Included the display of data frame fields on the alert event page.
- Modified Grafana utilities to incorporate transformations compatible with Grafana 11.5.2.
- Enhanced the build procedure and reduced the size of the engine's Docker image.
