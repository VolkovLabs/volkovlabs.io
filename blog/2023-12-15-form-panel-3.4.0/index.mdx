# Data Manipulation Panel 3.4.0

We are happy to announce the release of the Data Manipulation Panel 3.4.0. This release includes the following updates:

- Added ability to disable panel syncing for Initial Request.
- Added EventBus and AppEvents to Context parameter.
- Added replacing variables in Payload functions.
- Added compatibility with Grafana 10.2.2.

The plugin is under review.

## Getting Started

You can install the Data Manipulation Panel from the [Grafana Plugins catalog](https://grafana.com/grafana/plugins/volkovlabs-form-panel/) or using the Grafana command line tool.

For the latter, please use the following command:

```bash
grafana-cli plugins install volkovlabs-form-panel
```
