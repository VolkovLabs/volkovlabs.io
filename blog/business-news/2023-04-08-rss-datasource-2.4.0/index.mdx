# URL parameters, variables, and Grafana compatibility updates in Business News 2.4.0

We are happy to announce the release of the Business News Data Source 2.4.0. This release includes the following updates:

- Added URL parameters with dashboard variables support.
- Updated to the latest Grafana 9.4.7 toolkit and workflows.

## URL parameters

URL parameters allow reusing the data source for news feeds with the same URL.

For example, a data source can be set to YouTube. You can display specific or multiple news feeds using a dashboard variable.
