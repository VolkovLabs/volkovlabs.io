# Nested objects, pagination options, and E2E tests in Business Table 1.4.0

We're excited to announce the release of the Business Table panel 1.4.0. This feature release includes the following updates:

- Nested objects
- Show and hide configured columns
- Update pagination options
- Update aggregated cells
- Added E2E tests
