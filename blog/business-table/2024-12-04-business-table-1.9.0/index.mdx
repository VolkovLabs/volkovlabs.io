# Add and delete rows, new editors, cell types, and Grafana 11.3 Scenes updates in Business Table 1.9.0

We are excited to announce the release of the Business Table panel 1.9.0. This feature release includes the following updates:

- Added functionality to add and delete row.
- Added Image cell type.
- Added preformatted cell type.
- Added boolean cell type.
- Added default pagination size.
- Added support variables in data sources for editable and nested objects.
- Added custom value to editable select field.
- Added Sanitized HTML and Markdown column type.
- Added option to hide table header.
- Added Textarea column editor type.
- Compatibility with Grafana 11.3 and Scenes-powered dashboards.
