# Base64 Image/Video/Audio/PDF Panel 3.5.0

We are excited to announce the release of Base64 Image/Video/Audio/PDF Panel 3.5.0. This maintenance release includes the following updates:

- Removed the panel padding, allowing you to use the entire area of the panel.
- Updated workflows and compatibility with the latest Grafana 9.4.3 toolkit.

:::info Grafana Plugins catalog
The plugin was updated in the Grafana Plugins catalog on March 8, 2023.
:::

## No Padding

Removed the panel padding, which allows you to use the entire panel area for displaying images and PDF or media files.

## MQTT Data source

Base64 Image/Video/Audio/PDF Panel works with any data source. You can also use it with the MQTT data source to display the live camera feed.

## Getting Started

You can install Base64 Image/Video/Audio/PDF Panel from the [Grafana Plugins catalog](https://grafana.com/grafana/plugins/volkovlabs-image-panel/) or using the Grafana command line tool.

For the latter, please use the following command:

```bash
grafana-cli plugins install volkovlabs-image-panel
```
