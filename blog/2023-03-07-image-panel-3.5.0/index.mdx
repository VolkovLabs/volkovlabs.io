# Base64 Image/Video/Audio/PDF Panel 3.5.0

We released a new version of Base64 Image/Video/Audio/PDF Panel 3.5.0. This maintenance release

- Removed panel padding allowing to use of panel's space corner to corner.
- Updated to the latest Grafana 9.4.3 toolkit and workflows.

:::info Grafana Catalog
The plugin was updated in Grafana Catalog on March 8, 2023.
:::

<!--truncate-->

## No Padding

Removed panel padding allows using of the panel space from corner to corner and displays images, pdf and media files in the full width and height of the panel.

## MQTT Data source

The Base64 panel works with any data source. It was successfully used with the MQTT data source to view the live camera feed by the community.

## Getting Started

Base64 Image/Video/Audio/PDF visualization panel can be installed from the [Grafana Catalog](https://grafana.com/grafana/plugins/volkovlabs-image-panel/) or utilizing the Grafana command line tool.

For the latter, use the following command.

```bash
grafana-cli plugins install volkovlabs-image-panel
```
