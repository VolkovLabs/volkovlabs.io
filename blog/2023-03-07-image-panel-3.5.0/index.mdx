# Business Media Panel 3.5.0

We are excited to announce the release of the Business Media Panel 3.5.0. This maintenance release includes the following updates:

- Removed the panel padding, allowing you to use the entire area of the panel.
- Updated workflows and compatibility with the latest Grafana 9.4.3 toolkit.

:::info Grafana Plugins catalog
The plugin was updated in the Grafana Plugins catalog on March 8, 2023.
:::

## No Padding

Removed the panel padding, which allows you to use the entire panel area for displaying images and PDF or media files.

## MQTT Data source

Business Media Panel works with any data source. You can also use it with the MQTT data source to display the live camera feed.

## Tutorial

Business Media Panel is a Grafana plugin that renders PNG, JPG, GIF, MP4, WEBM, MP3, OGG, and PDF files encoded in the Base64 format.
