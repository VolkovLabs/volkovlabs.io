# Show variable in browser tab title, Date Time selectors, and Grafana 11.3 Scenes updates in Business Variable 3.7.0

We are happy to announce the release of the Business Variable panel 3.7.0. This release includes the following updates:

- Added advanced option to change browser tab title
- Added Date and Date Time selectors
- Compatibility with Grafana 11.4 and Scenes-powered dashboards.
