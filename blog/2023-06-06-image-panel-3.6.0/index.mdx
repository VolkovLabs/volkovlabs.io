# Base64 Image/Video/Audio/PDF Panel 3.6.0 supports Grafana 10

We released a new version of Base64 Image/Video/Audio/PDF Panel 3.6.0. This feature release

- Tested with Grafana 10 Preview.
- Added Toolbar to download, zoom and navigate between images.
- 100% Test coverage and added end-to-end test workflow.
- Migrated to Plugin Tools 1.5.2 with Node 18 and npm.

:::info Grafana Catalog
The plugin was updated in Grafana Catalog on June 15, 2023.
:::

## Image Toolbar

The new toolbar allows downloading, zooming and navigating between multiple images returned from the data source.

The toolbar is enabled in panel options by default, but buttons to display are not selected. Until buttons are selected, the toolbar is hidden.

## Getting Started

Base64 Image/Video/Audio/PDF visualization panel can be installed from the [Grafana Catalog](https://grafana.com/grafana/plugins/volkovlabs-image-panel/) or utilizing the Grafana command line tool.

For the latter, use the following command.

```bash
grafana-cli plugins install volkovlabs-image-panel
```
