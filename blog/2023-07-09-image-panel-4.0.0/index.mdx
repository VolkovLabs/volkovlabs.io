# Base64 Image/Video/Audio/PDF Panel 4.0.0

We released a new version of Base64 Image/Video/Audio/PDF Panel 4.0.0. This major release

- Requires Grafana 9 and Grafana 10.
- Added pan and pinch zoom in addition to the full-screen zoom, introduced in 3.6.0.
- Added description labels for images and media files.
- Updated to Grafana 10.0.0 and removed Grafana 8.5 support.

:::info Grafana Catalog
The plugin was updated in Grafana Catalog on June 28, 2023.
:::

## Pan and pinch-zoom

The pan and pinch zoom allows you to zoom in, and out and reset while working with high-resolution images.

When selecting the Zoom button to display on the toolbar you can choose between Full Screen and Pan and Pinch modes.

## Description

We added an option to specify a description field to add a label to the selected image. Label displayed on the bottom.

The image is auto-scaled when using Panel's height option. If you are using the original height, then the panel should be adjusted manually.

## Getting Started

Base64 Image/Video/Audio/PDF visualization panel can be installed from the [Grafana Catalog](https://grafana.com/grafana/plugins/volkovlabs-image-panel/) or utilizing the Grafana command line tool.

For the latter, use the following command.

```bash
grafana-cli plugins install volkovlabs-image-panel
```
