## Getting Started

**Business Intelligence Platform** is a powerful solution that harnesses **Docker containers** to deliver a modular, scalable, and user-friendly environment for alert-driven analytics. Whether you're just starting out or are an experienced user, our **[Quick Start Guide](/big/quick-start/)** is designed to help you set up and deploy the platform effortlessly.

This guide provides a step-by-step walkthrough of the essential setup process, ensuring you can get up and running in no time. Key topics include:

- **Configuring the [Business Engine](/big/engine/)**: Understand how to set up the core component that powers your analytics.
- **Launching the [Business Studio](/big/studio/)**: Deploy and access the intuitive interface on your local machine for seamless management and visualization.
