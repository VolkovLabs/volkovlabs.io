# Business Studio

Business Studio serves as the centralized control hub for the Business Intelligence platform, offering an intuitive interface that consolidates all essential configurations. It enables connections to multiple Business Engines—each linked to a Grafana instance—allowing authorized users to create and manage alert rules and actions efficiently.

<Image
  title="List of Configured Business Engines in Business Studio version 2.7.0"
  src="/img/big/business-studio/bs-main.png"
/>

## Downloading Business Studio

:::info Signed Releases
MacOS and Windows installers are signed, notarized, and encrypted. The end-to-end build process is fully automated via GitHub workflows to ensure security and integrity.
:::

Business Studio is a native application available for MacOS, Linux, and Windows. Download the latest version from the [Release Notes](../release).

## Auto-Update Feature

Business Studio supports automatic updates on MacOS and Windows, checking for new versions hourly on the GitHub repository. When an update is available, a dialog prompts users to restart and apply it.
